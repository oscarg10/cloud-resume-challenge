# German Learning App

The **German Learning App** is an interactive language-learning platform designed to help users practice German through structured exercises and intuitive feedback.

## What it does

- Provides sentence-based translation and transcription exercises  
- Supports progressive difficulty levels  
- Designed for extensibility with AI-driven features  
- Clean, user-friendly interface for daily practice

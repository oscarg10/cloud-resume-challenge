# Cloud Resume Challenge

Cloud serverless resume website built for the Cloud Resume Challenge.

## Live Sites

AWS: www.oscargresume.it.com

## Architecture

AWS: CloudFormation, S3, CloudFront, Lambda, DynamoDB, Route 53

Ansible is used for deployment automation.
